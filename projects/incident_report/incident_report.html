Incident Report

The Incident Report feature is designed to document daily incidents that may occur in the workplace and enable the appropriate response to reported incidents. Pages have been developed for both the mobile application and web portal. Through the mobile app, users can report an incident directly to their officer using the designated page. Upon submission of the form, the incident will be forwarded to the admin user portal. The current geolocation will be tracked at the time of submission. Incidents reported by users are displayed in the admin user portal, where they can be viewed and addressed accordingly. Actions can be taken on these reports, and the status of each action can be updated as necessary.
